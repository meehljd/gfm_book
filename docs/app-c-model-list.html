C.1 Category Definitions

  • DNA LM: DNA language models using self-supervised pretraining on genomic sequences
  • PLM: Protein language models trained on protein sequences
  • Seq→Func: Supervised sequence-to-function models predicting chromatin/expression from DNA
  • Splice: Specialized splice site prediction models
  • VEP: Variant effect predictors (various paradigms)
  • GFM: Genomic foundation model (broad, reusable representations)
  • PGS: Polygenic score or risk prediction models
  • GNN: Graph neural network for gene/pathway analysis
